# 科技创新强国之路 - 网站说明文档

## 1. 项目概述

**项目名称**：科技创新强国之路  
**项目主题**：展示国家在航天、深海、生物、信息等关键领域的科技创新成果，弘扬科学家精神，并提供互动交流入口。  
**设计风格**：现代科技风，以深蓝色（#0b3c6f）为主色调，搭配明亮辅助色（#ffd04b），强调简洁、专业与视觉冲击力。

---

## 2. 页面结构与主题

本项目包含 1 个首页和 5 个子页面，整体采用响应式布局，适配 PC、平板和移动端设备。

### 2.1 首页 (`index.html`)
作为网站的门户，首页集中展示了核心数据和关键板块。

*   **头部导航 (Header)**：
    *   包含网站 Logo 和标题。
    *   **导航菜单**：支持多级下拉菜单（“科技领域”），鼠标悬浮显示子菜单。
    *   **交互效果**：支持 Sticky 吸顶效果，页面滚动时始终固定在顶部。
*   **轮播图 (Slider)**：
    *   全宽展示航天、深海等领域的高清大图。
    *   包含左右切换按钮（悬浮显示）和圆角容器设计。
*   **主体内容区 (Main)**：
    *   **创新成果速览**：使用 **Flexbox 布局**，横向展示科研项目、重大装置等统计数字，悬浮有高亮效果。
    *   **科学家精神**：使用 **Flexbox 布局**，图文结合（左图右文），图片统一尺寸并添加圆角阴影。
    *   **前沿技术展望**：使用 **Grid 布局** (2x2)，展示量子计算、AI、6G、清洁能源等卡片，配有图标装饰，移动端自动切换为单列。
    *   **专题速览**：使用 **Float 布局**，左右两栏设计，引导用户进入具体专题页。
    *   **创新发展路径**：使用 **Position 绝对定位**，绘制垂直时间轴，展示从基础研究到应用推广的历程。
*   **侧边栏 (Aside)**：展示科技快讯列表和快速入口链接。
*   **页脚 (Footer)**：包含版权信息，使用 Sticky Footer 技术确保内容较少时也能贴在页面底部。

### 2.2 专题子页面 (`pages/`)
详细介绍各个科技领域的具体成就。

*   **包含页面**：
    *   `space.html` (航天强国)
    *   `deepsea.html` (深海探索)
    *   `bio.html` (生物科技)
    *   `info.html` (信息科技)
*   **结构**：
    *   复用公共头部和页脚。
    *   采用左右布局（响应式下变为单列），左侧为详细文字介绍和核心成就列表，右侧为相关配图。

### 2.3 加入创新 (`pages/join.html`)
提供用户互动和加入团队的通道。

*   **结构**：
    *   **表单区域**：居中卡片式布局，背景纯白带阴影。
    *   **表单项**：包含姓名、邮箱、兴趣领域（下拉菜单）、身份（单选按钮）、技能（多选框）、留言（文本域）。
    *   **样式**：输入框和按钮经过深度美化，具有聚焦高亮和悬浮上浮等微交互效果。

## 3. 技术栈与实现细节

本项目使用纯原生前端技术栈开发，无第三方库依赖。

### 3.1 HTML5
*   使用语义化标签构建页面结构：`<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<aside>`, `<footer>`。
*   使用语义化表单控件：`<input type="email">`, `<select>`, `<textarea>`, `<label>`。

### 3.2 CSS3
*   **布局技术**：
    *   **Flexbox**：用于导航栏、统计卡片、图文板块、表单行布局。
    *   **Grid**：用于“前沿技术展望”板块的网格布局。
    *   **Float**：用于“专题速览”板块的传统布局演示。
    *   **Position**：用于 Sticky 导航栏、轮播图按钮定位、时间轴节点定位。
*   **响应式设计 (Responsive Web Design)**：
    *   使用 `<meta name="viewport">` 控制视口。
    *   使用 `@media` 媒体查询，针对平板 (max-width: 768px) 和手机 (max-width: 480px) 调整布局（如将多列变为单列、调整导航栏样式）。
    *   图片自适应：`max-width: 100%`, `object-fit: cover`。
*   **视觉增强**：
    *   `box-shadow`: 为卡片和容器添加层次感阴影。
    *   `border-radius`: 统一的圆角设计风格。
    *   `linear-gradient`: 按钮渐变背景。
    *   `backdrop-filter`: 轮播图按钮的毛玻璃效果。
    *   `transform` & `transition`: 鼠标悬浮时的位移、旋转和颜色平滑过渡。

### 3.3 JavaScript
*   **DOM 操作**：实现轮播图图片的切换逻辑。
*   **事件监听**：处理点击事件（上一张/下一张）和自动播放定时器。
